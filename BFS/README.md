# BFS 广度优先搜索

Word ladder (单词变换问题)
* 给定字典和一个起点单词、一个终点单词，每次只能变换一个字母，问从起点单词是否可以到达终点单词？最短多少步？
* 如:
* start= "hit"
* end = "cog"
* dict = ["hot","dot","dog","lot","log"]
* "hit" -> "hot" -> "dot" -> "dog" -> "cog"
