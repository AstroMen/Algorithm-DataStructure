ArticulationPoint
* 给定某无向连通图G，若删除某结点X已经  与X相邻接的所有边，图G变成非连通图，  则结点X称为图G的割点。
* 问：给定某图的邻接矩阵或邻接表，如何计算该图的所有割点？
